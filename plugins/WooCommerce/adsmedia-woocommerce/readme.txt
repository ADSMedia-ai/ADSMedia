Send WooCommerce transactional emails via ADSMedia API. High deliverability for order confirmations, shipping notifications, and more.

== Description ==

ADSMedia for WooCommerce replaces the default WooCommerce email sending with ADSMedia's high-deliverability email API.

**Features:**

* Send all WooCommerce emails through ADSMedia
* Choose which email types to send
* Test email functionality
* Resend order emails
* Daily email statistics
* Full HPOS compatibility
* Logging support

**Supported Email Types:**

* New Order (Admin)
* Cancelled Order (Admin)
* Failed Order (Admin)
* Order On-Hold
* Processing Order
* Completed Order
* Refunded Order
* Customer Invoice
* Customer Note
* Reset Password
* New Account

== Installation ==

1. Upload `adsmedia-woocommerce` folder to `/wp-content/plugins/`
2. Activate the plugin
3. Go to WooCommerce > ADSMedia Email
4. Enter your API key
5. Enable the plugin and select email types
